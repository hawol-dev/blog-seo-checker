### 1. AI 콘텐츠 생성

#### 1-1. WriteFlow — 블로그 AI 라이터

> 주제 입력 → AI가 아웃라인 → 초안 → 편집까지. SEO 점수 실시간 표시

- **스택**: Next.js + Express + AI API + Redis(캐싱) + PostgreSQL
- **수익 모델**: 무료 월 5편 → **$19/월** Pro (무제한 + 톤 커스텀)
- **핵심 기능**:
  - 주제 → 아웃라인 → 초안 3단계 워크플로우
  - SEO 점수 실시간 사이드바 (키워드 밀도, 가독성, 메타태그)
  - 브랜드 톤/보이스 프리셋 저장
  - 히스토리 관리 + 버전 비교
- **차별점**: 범용 AI 글쓰기가 아닌 **블로그 포스트 전문 파이프라인**

#### 1-2. CopyDesk — 마케팅 카피 AI

> 광고 문구 / 랜딩페이지 카피 / 제품 설명문 / 이메일 제목 AI 생성

- **스택**: Next.js + NestJS + AI API + Redis + PostgreSQL
- **수익 모델**: 무료 월 20생성 → **$14.99/월** Pro
- **핵심 기능**:
  - 카피 유형별 특화 템플릿 (광고/랜딩/이메일/SNS)
  - A/B 테스트용 문구 변형 자동 생성
  - 카피라이팅 프레임워크 내장 (AIDA, PAS, BAB)
  - 생성 히스토리 + 즐겨찾기 폴더
- **차별점**: 범용이 아닌 **마케팅 카피 전문**, 프레임워크 기반 구조화

#### 1-3. ThreadCraft — SNS 쓰레드 생성기

> 긴 글/아이디어 입력 → X(트위터) 쓰레드, 링크드인 포스트로 자동 분할 + 포맷팅

- **스택**: Next.js + Express + AI API + Redis
- **수익 모델**: 무료 월 10개 → **$9.99/월** (무제한 + 바이럴 패턴 분석)
- **핵심 기능**:
  - 긴 글 → 쓰레드 자동 분할 (글자 수 최적화)
  - 바이럴 후킹 문장 AI 추천
  - 플랫폼별 포맷 자동 변환 (X/LinkedIn/블로그)
  - 드래프트 저장 + 예약 발행 메모
- **차별점**: **SNS 쓰레드 특화** — 블로그가 아닌 짧은 시리얼 콘텐츠

### 2. AI 학습 코치

#### 2-1. StudyForge — AI 커리큘럼 빌더

> 학습 목표 입력 → AI가 주차별 커리큘럼 생성 + 웹 리소스 큐레이션 + 진도 관리

- **스택**: Next.js + NestJS + AI API + Redis + PostgreSQL + Cron
- **수익 모델**: 무료 1과목 → **$7.99/월** 무제한
- **핵심 기능**:
  - "3개월 안에 React 마스터" → 주차별 학습 플랜 자동 생성
  - 무료 웹 리소스 (유튜브/문서/블로그) 자동 큐레이션
  - 주차별 체크리스트 + 완료율 대시보드
  - 진도 지연 시 AI 플랜 자동 조정
- **차별점**: 유료 강의 없이 **무료 웹 리소스만으로** 커리큘럼 구성

#### 2-2. FlashAI — AI 플래시카드 생성기

> 텍스트/PDF/URL 입력 → AI가 핵심 개념 추출 → 플래시카드 자동 생성 + 반복 학습 스케줄

- **스택**: Next.js + Express + AI API + Redis + PDF Parser + PostgreSQL
- **수익 모델**: 무료 50장 → **$4.99/월** 무제한
- **핵심 기능**:
  - 텍스트/PDF/URL 아무거나 넣으면 카드 자동 생성
  - 스페이싱 이펙트(간격 반복) 기반 복습 스케줄
  - 카드 난이도별 분류 (쉬움/보통/어려움)
  - 데크 공유 + 커뮤니티 덱 마켓
- **차별점**: Anki의 현대적 대안 — **AI 자동 생성 + 깔끔한 웹 UI**

#### 2-3. ExamCraft — AI 모의고사 생성기

> 과목/범위/난이도 입력 → AI가 모의고사 생성 + 자동 채점 + 취약점 분석

- **스택**: Next.js + NestJS + AI API + Redis + PostgreSQL + Chart.js
- **수익 모델**: 무료 주 1회 → **$6.99/월** 무제한
- **핵심 기능**:
  - 객관식/주관식/서술형 혼합 출제
  - 자동 채점 + 오답 해설 AI 생성
  - 취약점 레이더 차트 + 보강 추천
  - 과목별 점수 추이 그래프
- **차별점**: **시험 준비 특화** — 학습이 아닌 평가/진단에 집중

### 3. 북마크/지식 관리

#### 3-1. LinkBrain — AI 스마트 북마크

> 브라우저 확장으로 원클릭 저장 → AI 자동 태그/분류 → 시맨틱 검색

- **스택**: Next.js + Express + AI API + Redis + PostgreSQL + Chrome Extension
- **수익 모델**: 무료 100개 → **$4.99/월** 무제한
- **핵심 기능**:
  - 원클릭 저장 (브라우저 확장)
  - AI 자동 태그 + 카테고리 분류
  - "지난달 저장한 React 관련 글" 자연어 검색
  - 중복 감지 + 죽은 링크 알림
- **차별점**: 기존 북마크의 문제 = **못 찾음**. AI 시맨틱 검색으로 해결

#### 3-2. ReadLater Pro — AI 요약 리딩 리스트

> 나중에 읽기 저장 → AI 자동 요약 → 주간 다이제스트 리포트

- **스택**: Next.js + NestJS + AI API + Redis + Cron + PostgreSQL
- **수익 모델**: 무료 월 20개 → **$3.99/월** 무제한
- **핵심 기능**:
  - URL 저장 시 AI 3줄 요약 자동 생성
  - 읽기 예상 시간 표시
  - 주간 다이제스트 (저장했지만 안 읽은 글 리마인드)
  - 하이라이트 + 메모 기능
- **차별점**: "저장만 하고 안 읽는" 문제를 **AI 요약 + 리마인드**로 해결

#### 3-3. ResearchBoard — 리서치 특화 지식 보드

> 주제별 보드에 링크/메모/스크랩 정리 → AI가 보드 내용 종합 인사이트 생성

- **스택**: Next.js + Express + AI API + Redis + PostgreSQL + S3
- **수익 모델**: 무료 3보드 → **$5.99/월** 무제한
- **핵심 기능**:
  - 핀터레스트식 보드 + 카드 레이아웃
  - 링크/텍스트 메모/이미지 스크랩 혼합 저장
  - "이 보드 내용을 요약해줘" AI 종합 분석
  - 보드 공유 (읽기 전용 퍼블릭 링크)
- **차별점**: 북마크가 아닌 **리서치 프로젝트 단위** 관리

- **스택**: Next.js + Express + AI API + Redis + PostgreSQL

  - 생성 결과 실시간 미리보기 + 편집
  - 변환 히스토리 + 즐겨찾기
- **차별점**: "하나만 쓰면 전부 다" — **시간 절약이 핵심 가치**

#### 4-2. SlideForge — 텍스트 → 프레젠테이션 변환기

> 블로그 글/문서/메모 입력 → AI가 슬라이드 덱 자동 생성

- **스택**: Next.js + NestJS + AI API + Canvas/SVG + Redis + PostgreSQL
- **수익 모델**: 무료 월 2개 → **$9.99/월** 무제한
- **핵심 기능**:
  - 텍스트 → 슬라이드 자동 분할 + 레이아웃
  - 디자인 테마 프리셋 (미니멀/비즈니스/크리에이티브)
  - 차트/다이어그램 자동 생성
  - PDF/PPTX 내보내기
- **차별점**: Canva/파워포인트가 아닌 **"글만 쓰면 발표자료 끝"**

#### 4-3. ClipScript — 영상 → 텍스트 콘텐츠 변환기

- **스택**: Next.js + Express + YouTube Transcript API + AI API + Redis
- **수익 모델**: 무료 월 5개 → **$7.99/월** 무제한
- **핵심 기능**:
  - YouTube URL만 입력하면 자막 자동 추출
  - 자막 → 구조화된 블로그 글 변환
  - 핵심 요약 / 쓰레드 / 뉴스레터 포맷 선택
  - 타임스탬프 기반 인용 링크 생성
- **차별점**: **영상 → 텍스트 방향** 리퍼포징 특화
